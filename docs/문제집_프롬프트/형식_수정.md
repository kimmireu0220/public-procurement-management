# 형식 일괄 수정 프롬프트 (전 과목)

기존 `output/agent_extract/<slug>/partN.md`의 **형식만** 고칠 때 사용. 문항 내용·번호·정답은 바꾸지 않는다.

```
너는 `output/agent_extract/〈slug〉/part〈N〉.md`의 **형식만** 수정하는 편집자다.
문항 내용·번호·정답은 바꾸지 않는다.

## 작업
1. 객관식: `N. 문제? ① … ② … ③ … ④ …` 한 줄 형태를 찾아
   - `N. 문제?` 한 줄
   - `   ① …` / `   ② …` / `   ③ …` / `   ④ …` 각각 한 줄로 분리 (3칸 들여쓰기)
2. 4과목(`4과목_공공조달 관리실무`)만:
   - 문항을 읽고 유용한 암기팁이 있으면 `> 💡 암기팁: …` 추가 (source 직전)
   - 적용 어려우면 생략. 정답 번호·해설은 넣지 말 것
3. 수정 후 `python3 tools/build_problem_book.py --subject 〈과목번호〉` 실행

## 금지
- 정답 섹션 수정
- 문항 삭제·추가
- 1~3과목에 암기팁 추가

## 자가검사
- 문항 본문 줄에 `①.*②` 패턴이 남아 있으면 선지 분리 미완료
```

규칙 상세: [`extraction_guide.md`](../extraction_guide.md)
