# 시험 모의 선별 프롬프트

`output/problem_book_final/` 문제 은행(3,325문항)에서 **실제 1회 시험 규모**로 모의 세트를 뽑을 때 사용한다.  
추출·형식 규칙: [`extraction_guide.md`](../extraction_guide.md)

## 시험 규모

**통합 기준:** [`시험_안내.md`](../시험_안내.md)

| 구분 | 과목 | 문항 수 | 시간·형식 |
|------|------|--------:|-----------|
| 필기 | 1 공공조달과 법제도 이해 | **30** | CBT 4지선다 · **필기 합계 80문항 · 2시간** |
| 필기 | 2 공공조달계획 수립 및 분석 | **20** | 동일 |
| 필기 | 3 공공계약관리 | **30** | 동일 |
| 실기 | 4 공공조달 관리실무 | **20 내외** | 필답형 · **2시간 30분** |

**일정 (공고 제2025-209호):** 필기 10.3(토) · 실기 11.14(토) — `sources/공식_qnet_시행공고/`

## A. 필기 모의 (과목별 문항 수 상이)

작업 시 **〈N〉=과목번호**, **〈M〉=해당 과목 문항 수**(1·3과목 30, 2과목 20)만 교체한다.

```
너는 공공조달관리사 1회 필기 〈N〉과목 **모의 출제자**다.
`output/problem_book_final/〈slug〉/parts_clean/part*.md`에서 **〈M〉문항**을 선별해 모의시험지를 만든다.

## 과목별 목표 문항 수
- 1과목: 30문항
- 2과목: 20문항
- 3과목: 30문항

## 선별 우선순위 (높은 순)
1. `단원별 출제예상문제` — 4지선다
2. ㉠㉡·ㄱㄴㄷ **복수선택**·`( )` **빈칸**형 — Q-Net 예제 유형 (`sources/공식_qnet_예제문제/`)
3. `Check Q&A` — 개념 확인용 (전체의 20% 이하)
4. **제외**: `(O/X)` 문항 — 실제 필기에 없음

## 구성 원칙
- Part·Chapter 골고루 (한 Part에 〈M〉문항 몰지 않음)
- 1과목 Intro `page_0002.jpg` 출제기준 **(30)/(20) 표기** 비중 참고 (2과목은 20문항 분량에 맞게 핵심 단원 우선)
- 동일 페이지·유사 지문 중복 금지
- 정답은 별도 `모의〈N〉과목_정답.md` (문항 번호 1~〈M〉 재부여)

## 출력 형식
- 문항 번호 1~〈M〉 연속
- 선지 줄바꿈·㉠㉡ 소항목 분리는 extraction_guide 준수
- 각 문항 끝: `<!-- source: 원본 Part/페이지 -->` 유지

## 완료 보고
Part·Chapter별 문항 수, 제외한 OX 수, ㉠㉡·빈칸형 포함 개수.
```

### 필기 과목별 첨부·목표

| 과목 | 문항 | parts_clean 경로 |
|------|-----:|------------------|
| 1 | 30 | `output/problem_book_final/1과목_공공조달의 이해/parts_clean/` |
| 2 | **20** | `output/problem_book_final/2과목_공공조달 계획분석/parts_clean/` |
| 3 | 30 | `output/problem_book_final/3과목_공공계약관리/parts_clean/` |

## B. 실기 모의 (20문항)

```
너는 공공조달관리사 1회 실기 **모의 출제자**다.
`output/problem_book_final/4과목_공공조달 관리실무/parts_clean/part*.md`에서 **20문항**을 선별한다.

## 선별 우선순위
1. `서술형 출제예상문제` — 10~12문항 (절차·이유·차이 설명)
2. `핵심 최종점검` — 6~8문항 (단답·키워드)
3. `Check Q&A` — 2~4문항 (실무 판단형만)
4. Part 1~8 골고루 — 입찰준비→전자조달 흐름 유지 (4과목 Part 1 `page_0002.jpg` 출제기준 8개 주요항목 참고)

## 구성 원칙
- 한 주제에 3문항 이상 연속 금지
- 4과목 암기팁(`> 💡`)은 모의지에 **포함하지 않음** (학습용)
- 채점 루브릭: 정답 섹션 키워드·절차 순서 기준 1줄 요약

## 출력
- `모의실기_문제.md` (20문항, 1~20)
- `모의실기_정답.md` (키워드·채점 포인트)
```

## C. 검수 (모의 세트 완성 후)

1. 문항 수 = 목표 (1·3과목 30, 2·4과목 20)
2. OX 혼입 없음 (필기)
3. 선지·㉠㉡ 줄 분리 (`fix_choice_lines.py` 또는 수동)
4. 원본 source 주석 유지
5. (선택) Q-Net 예제 6문항·Intro 출제기준과 단원 분포 대조
