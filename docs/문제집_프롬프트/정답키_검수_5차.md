## SOP (문항 1건)

1. `partN.md` 본문에서 문항 블록(지문+선지 또는 O/X) 읽기
2. `# Part N 정답` 이후 동일 section·번호의 정답·해설 매칭
3. `<!-- source: Part N/page_XXXX.jpg -->` → `output/ocr/<slug>/Part N/page_XXXX.txt` 대조
4. source 없으면 인접 OCR에서 지문 매칭
5. **도메인 지식**으로 정답 후보 재도출(선지별 O/X)
6. extract 정답·해설과 비교 → 3축 판정
7. 과목별 `검수보고.md`에 P0/P1/P2·Part 통계 기록

### 판단 우선순위

1. 원본 JPG / OCR 지문·해설
2. 법령·시행령·조달청 고시
3. 조달청 표준교재 · Q-Net 예제
4. extract (합성 문항은 OCR 불일치만으로 PASS 금지)
