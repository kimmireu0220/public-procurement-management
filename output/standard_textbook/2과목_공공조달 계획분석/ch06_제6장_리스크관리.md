# 제6장 공공조달 리스크 관리 (표준교재 핵심)

> PDF p.245–279 · 박문각 Part 4

## 한눈에 보기

- **리스크** = 목표 달성에 대한 **불확실성의 영향** (ISO 31000).
- **리스크 관리 = 조달계획부터**(표 **47**) — 입찰·계약만이 아님; 요구·범위 불명확이 선행 위험.
- **ISO 31000**: 의사소통 → 범위·기준 → **식별·분석·평가** → 처리 → 모니터링 → 기록.
- **처리 5R**: 회피 · 감소 · 이전 · 수용 · (공유).
- **조달 리스크 영향**: ①총비용↑ ②납기지연 ③품질↓ → ②차: 성과·신뢰·평판.
- **단계별 표 46~54**: 계획·시장조사·전략·입찰문서·설명·접수·평가·계약·관리.

---

## 절별 핵심

### 제1절 리스크 관리 개요

**ISO 31000:2018**

- **정의**: 기대 결과에 관한 **불확실성의 영향**.
- **8원칙**: 가치 창출·보호 · 전 프로세스 필수 · 의사결정 일부 · 불확실성 명시 · 체계·시의 · 정보 기반 · 투명·포괄 · 역동·반복 · **지속 개선**.

**6모듈 프로세스** (그림 70)

1. **의사소통·컨설팅** (전 단계)
2. **범위·조직상황·기준** — 허용·불허 수준
3. **리스크 평가**: 식별 → 분석 → 평가
4. **리스크 처리**
5. **모니터링·검토**
6. **기록·보고**

**1.3 평가**

- **식별**: 유무형·원인·위협·기회·약점·강점
- **분석**: 심각성×가능성; FMEA·델파이·의사결정트리
- **평가**: 기준 대비 우선순위

**1.4 처리**

| 대안 | 내용 |
|------|------|
| **회피** | 활동 자체 중단 |
| **감소** | 빈도·피해 완화 |
| **이전** | 보험·계약 |
| **수용** | 감수 |

**1.5~1.6**: 모니터링·기록 — **RMM(위험관리매트릭스)** 정기 갱신.

---

### 제2절 공공조달 리스크 관리

**2.1 조달 위험**

- 단계마다 **고유 위험**; 조기 식별 → **사양·계약조건 개정**·RFI·Q&A.
- **1차 영향 3**: 비용 · 일정 · 품질.
- **2차**: 성과 · 전문성 불신 · **정치·평판**.
- **조달계획 단계**부터 식별·분석·평가·처리·**담당자 배정**.

**2.2 프로세스**

- 상황 설정 → 식별 → 분석 → **우선순위** → 처리.

**표 42 — 발생 가능성 (3년)**

| 수준 | |
|------|--|
| A | 아주 높음 |
| B | 높음 |
| C | 낮음 |
| D | 매우 낮음 |

**표 43 — 영향 4단계**: Minor · Moderate · Major · **Severe**.

**표 44 — 위험 수준**: 확률×영향 → Low · Medium · High · **Critical**.

**허용 기준 예**: 재정손실 · 일정지연 · **구매사양 오차** · 품질 · 사업지연.

**표 45 — 처리 대안**: 회피·완화·이전·수용·공유.

**표 46 — 일반적 조달 위험**: 발생 가능성·완화(경쟁·투명·VFM·감사 등).

### 제3절 단계별 리스크 관리

| 표 | 단계 | 대표 위험 | 대응 |
|----|------|-----------|------|
| **47** | 조달계획 | 요구 불명확·범위 | 이해관계자·SOW·우선순위 |
| **48** | 시장조사 | 정보 편향·RFI 남용 | 다원 정보원·공정 RFI |
| **49** | 조달전략 | 방법 오선택 | 법·정책·유형별 검토 |
| **50** | 입찰문서·승인 | 사양 오류·승인 지연 | 체크리스트·내부검토 |
| **51** | 설명·접수 | 불공정 Q&A·마감 | 공개 답변·시스템 |
| **52** | 평가 | 편향·유출 | 위원 제척·기밀 |
| **53** | 계약 확정 | 결격·협상 결렬 | 적격 재확인·차순위 |
| **54** | 계약관리 | 이행·변경·분쟁 | CMP·모니터링·에스컬레이션 |

**공통**: **입찰 전·초기** 개선이 비용 최소 — 계획·시장조사·사전규격.

## 시험 출제 포인트

- **ISO 31000 정의·8원칙·6모듈**.
- **5R 처리** — 회피 vs 감소 vs 이전.
- **표 42~44** — A~D · 4단계 영향 · Critical.
- **1차 3·2차 3** 영향 영역.
- **단계별 표 47~54** — 어느 단계 위험인지.
- **조달계획 단계**부터 리스크 — **입찰만이 아님**.
- RMM **정기 갱신**.

## 함정·비교형

| A | B | 포인트 |
|---|---|--------|
| 회피 | 감소 | 중단 vs 완화 |
| 이전 | 수용 | 보험 vs 감수 |
| 1차 | 2차 | 비용·일정 vs 평판 |
| 식별 | 분석 | 목록 vs 우선순위 |
| 조달계획 | 계약관리 | 표 47 vs 54 |
| RFI 편향 | 공정 RFI | 표 48 |
| 평가 유출 | 제척 | 표 52 |
| ISO 31000 | 조달 5단계 | 국제 vs 업무 |
| **ISO 31000** | **국가계약법** | 국제 **리스크 원칙** vs 국내 **입찰·계약 절차** 법령 |
