# 제2장 공공조달 입찰 실행절차 (표준교재 핵심)

> PDF p.51–94 · 박문각 Part 2

## 한눈에 보기

- **4절 구조**: 입찰공고(Solicitation) → 평가(Evaluation) → 낙찰자선정 검토·승인 → 계약체결.
- **발주 9단계**(표 18): 발주계획 → 계약심사·일상감사 → 결과통보·확정 → 사업추진계획 → 내부/조달청 의뢰 → **사전규격공개** → 입찰공고.
- **전주기 ~20절차**: 수요발생 → 조달요청 → 사전규격 → 구매결의 → 입찰공고 → 기초금액 → 입찰 → 예정가격 → 개찰 → 낙찰 → 계약 → 착수계 …
- **평가**: 최저가(Lowest Cost) vs 최고가치(Best Value); ITB/RFQ vs RFP; 2단계·적격심사.
- **낙찰**: **계약의 예약** — 계약 체결 ≠ 낙찰; 계약심의위원회·금액별 승인.
- **계약**: 보증금 10%(예외 5%) · 착수계 **10일** · 전자서명.

## 절별 핵심

### 제1절 입찰공고(Solicitation)

**1.1 발주와 입찰**

- 공급업체 관점 **입찰공고 = 조달 시작**이지만, 실제는 **사전규격·발주계획** 선행.
- **발주 9단계**(업무일 **~20일**, 입찰공고 전 **~1개월** 준비):

| 단계 | 내용 | 소요(예) |
|------|------|----------|
| 1 | 발주계획(예산·계약방법·낙찰방법·일정·납품기한) | 사전 |
| 2 | 계약심사 의뢰 | D+1 |
| 3~5 | 심사·감사 결과·확정 | D+3~8 |
| 6 | 사업추진계획 | D+11 |
| 7 | 내부계약의뢰(자체) | D+13 |
| 8 | 조달청 계약의뢰(중앙) | D+18 |
| 9 | 사전규격 등록 | D+23 |

**계약심사 중점**: 설계 불일치·표준시장단가·품셈·법정경비·물량·가격 — **낙찰 후 보증금 반환은 심사 대상 아님**.

**일상감사**: 합법성·타당성·예산·계약방법·예정가격 등.

**사전규격공개**

- 물품·용역 **의무**(공사도 관행); **제외**: 긴급·비밀·수의·5,000만 미만·연 1회 공개 품목.
- 기간: 일반 **5일** · 긴급 **3일** · SW **10일**.
- 수의계약 해당(국가계약법 시행령 **제26조**·지방 **제25조**) 시 입찰 대신 수의.

**구매결의서**: 구매관리번호·정부물품목록·계약방법·낙찰방식·예산·입찰일시·납품·검사검수기관·입찰유의서 등 — **검사·검수 합격 여부는 미포함**.

**입찰공고**(국가계약법 시행령 **제36조** **21항**)

**공고기간**: 입찰마감 **7일 전** 공고(국가); 협상 **40일 전**(예외 10일); 지방 추정가 **1억 미만 10일·1~10억 20일·10억↑ 40일**.

**기초금액**(시행령 **제9조**): 예정가격 **확정 전** 가격 — **협상 등 생략 가능**; 간행물은 **정가**.

**입찰·보증금**: 입찰참가 신청마감까지 **입찰금액 100분의 5** (면제 가능); **유찰 시 재공고 없이 재입찰**·타 업체 참가 가능.

**예정가격**: 기초금액 **±2%**(국가) **±3%**(지방) 범위 **15개** 예비가격 중 **4개 선택 최다 → 산술평균**.

**낙찰자 선정**: **계약의 예약** — 계약상대자 지위 아님.

**계약체결**: 보증금 **10%**(재난 등 **5%**); **착수계 10일** 이내.

**표 19·20**: 입찰문서·계약조건 구성(입찰유의서·청렴서약·특수조건·일반조건 등).

### 제2절 평가(Evaluation)

**평가 원칙**: 입찰공고 **사전 공개** 기준; VFM; **가격만이 아님** — TCO·품질·일정·규정·위험.

**표 21 — 평가위원회 권고**: 브리핑·이해상충 신고·기밀·공개 질의(조달담당 경유).

**평가위원회 규모**: 단순 **5명** · 복잡 **10명**; ESG 시 **외부 전문가** 필수 증가.

**(1) 최저가격 + 기술 충족 (ITB·RFQ)**

- **Lowest Cost** = 전통적; **Best Value**로 전환 추세.
- **하한선** 설정·**적격심사**·2단계로 보완.
- 수의: 예정가격·예산 **88~90%** 이상 최저가.

**(2) 가중치 평가 (RFP)**

- 기술·품질·일정·관리 + 가격; **가격 가중 10~30%**(단순 시 50%).
- **기술 최소점 미달 → 가격 무관 탈락**.
- 기술평가 **가격 이전** 수행.

**예비심사**: 합격/불합격; **중대 vs 기타 오류** 구분; 과거문서 보완 가능.

**평가 절차**: 형식·기술·재정(경제성)·업체적격·발표·**평가보고서**.

**협상**: RFP **일반적**; ITB **원칙 불가**(예외: 단독 최저가·과소가 → **최저가적정성심의**).

### 제3절 낙찰자선정 검토·승인

**목적 5가지**: 독립·공정 확인; 정책·절차 준수; 예산·경제성; VFM·사회적 가치; **당국 승인**.

**승인 체계**: 금액별 **조달담당→관리자→상위**; 대형·고파급 → **계약심의위원회**; RFQ·ITB는 **위임** 가능.

**확인 4항**: 법령 준수; 참가자격; 부적격·예외 기록; **이행능력·실사**.

---

### 제4절 계약체결(Contract)

**협상 주제**(표 22): 가격·일정·범위·품질·인력·하도급·지급조건 등.

**협상 준비**(표 23): BATNA·목표·한계·팀·문서.

**계약 성립**: 보증금·인지세 수납 → 계약서 **날인/전자서명**.

## 시험 출제 포인트

**발주·사전규격**

- **9단계 순서** — Check Q&A: 발주계획→내부심사→확정→사업추진→계약요청.
- 사전규격 **5/3/10일** · SW 10일 · 5,000만 제외.
- **입찰공고**: 국가 일반 **7일 전** · **협상 40일 전**(예외 **10일**) — 1장 10일·협상 기간 혼동 주의.
- **재공고 없이 재입찰** · 타 업체 참가 가능.

**금액·보증**

- **기초금액 = 예정가격 전** · 협상 생략 가능.
- 예정가격: **±2%·15개·4개·평균**(국가).
- 입찰보증 **5%** · 계약보증 **10%**(5% 조정) · 착수계 **10일**.

**평가·낙찰**

- **ITB = 최저가+기술충족** · **RFP = 가중+협상**.
- **낙찰 = 계약의 예약**.
- **유찰 = 입찰 미성립**.

**오답 유형**

- 「반드시 재공고」 — **오답**
- 「입찰보증 10%」 — **5%**
- 「기초금액 = 낙찰 후」 — **예정가격 전**

## 함정·비교형

| 비교 | A | B | 구분 |
|------|---|---|------|
| ITB | 가격 중심 | RFP | 표준화·최저가 |
| RFQ | 견적·소액 | RFP | 가장 단순 |
| Lowest Cost | 전통 | Best Value | 2000년대 전환 |
| 기초금액 | 예정가 전 | 예정가 | 협상 생략 |
| 낙찰 | 예약 | 계약 | 법적 지위 |
| 유찰 | 미성립 | 낙찰 | 재입찰 가능 |
| 검사 | 이행 | 검수 | 수량·손상 |
| 계약심사 | 입찰 전 | 낙찰 후 보증 반환 | 심사 범위 |
| 공고기간 | 7일(국가) | 40일(협상) | 예외 10일 |
| 2단계 1인 | 재공고 | 동시입찰 | 규격적격 1명 |
| 입찰공고 기간 | **10일**(1장 일반) | **7일**(국가 2장) | 시행령 제36조 · 협상 40일 |
| 예정가 범위 | **±2%**(국가) | **±3%**(지방) | 15개 중 4개 평균 |
