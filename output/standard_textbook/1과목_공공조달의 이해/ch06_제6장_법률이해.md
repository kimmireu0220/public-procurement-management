# 제6장 공공조달 법률 이해 (표준교재 핵심)

> PDF p.321–410 · 박문각 Part 6

## 한눈에 보기

- **법적 체계** — `국가계약법` + `지방계약법`(통칭 **공공계약법**) · `조달사업법`(조달청 중앙조달) · `전자조달법`(2013) · `공기업·준정부기관 계약사무규칙`(계약사무규칙).
- **공공계약법 성격** — 민법의 **특별법·절차법**; 국가·지자체가 **사경제 주체**로 체결하는 **사법상 계약**에 적용. **공법상 계약**(연구개발 협약 등)은 원칙적으로 **배제**.
- **국가계약법 핵심 축** — 계약 원칙(제5조) → 추정·예정가격 → 계약방법(일반·제한·지명·수의) → 낙찰제도 → 체결·이행 → 이의·분쟁조정.
- **민법 연계** — 전형계약(도급 중심), 청약·승낙·예약, **해제(소급)** vs **해지(장래)** , 도급 vs 위임, 화해 ↔ 국가계약분쟁조정.
- **조달사업법** — 조달청 **정책·중앙조달** 역할; 제3자 단가·다수공급자·대지급·불공정조달행위·거래정지.
- **전자조달법** — 전자공고·전자입찰·전자계약·하도급 전자처리·전자공개수의(안내공고·전자견적서).
- **국가 vs 지방** — 구조 유사, 지방은 조문·시행령 **세분·추가**(청렴서약제, 주민참여감독, 국제입찰 고시금액, 계약서 작성 기준 등).

## 절별 핵심 (제1절 국가·지방계약법)

### 1.1 법체계·적용범위

| 구분 | 내용 |
|------|------|
| **공공계약법** | `국가계약법` + `지방계약법` — 공공조달 실행·관리 **기준법** |
| **조달사업법** | 조달청 기능·역할(중앙조달·정책) — 전면 통합법 아님 |
| **전자조달법** | 2013 제정 — 전자조달 기반 운용 |
| **계약사무규칙** | 공기업·준정부기관 특성 반영 |
| **적용** | 사법상 계약 O / **별도 공법 근거 계약** X (국가계약법 제1·3·4조, 판례) |
| **특별법** | 국가계약법은 **민법에 우선**; 계약조건·특수조건 > 민법 (하자담보 등) |

### 1.2 계약의 원칙 (`국가계약법` 제5조)

| 원칙 | 요지 |
|------|------|
| **계약자유·신의성실** | 대등한 합의, 신의성실 이행 (제5조①) |
| **호혜** | GPA 가입국 국민·물품·용역 **차별 특약 금지** (제5조②) |
| **권한남용 금지** | **부당한 특약등** 설정 금지·**무효** (제5조③④) |
| **청렴** | **청렴계약** 의무; 위반 시 입·낙찰 취소·해제·해지 (제5조의2, 시행령 제4조의2) |

### 1.3 추정가격·예정가격

| 용어 | 정의·역할 |
|------|-----------|
| **추정가격** | 예정가격 전 산정; **부가세 제외**; 국제입찰·계약방법·공고기간·현장설명 기준 |
| **추정금액** | 추정가격 + 부가세 (+ 공사 관급자재) |
| **예정가격** | 낙찰·계약금액 **결정기준**; 추정가격 + 부가세 |
| **국제입찰** | 추정가격 ≥ **고시금액**(기재부 장관 고시) |
| **예정가격 결정순** | 거래실례 → 원가계산 → 표준시장단가 → 감정·유사·견적 |
| **복수예비가격** | 기초금액(추정+VAT) ±2% · 15개 생성 → **최다 선택 4개 산술평균** = 예정가격 |

**예정가격 작성 생략·미작성** (시행령 제7조의2): 일괄입찰·실시/기본설계 기술제안입찰 / 수의(견적 제출 시 제외)·협상·경쟁적 대화·**개산계약** 등.

### 1.4 계약방법·공급형태

**원칙:** `국가계약법` **제7조** — **일반경쟁**; 예외 — **제한·지명·수의**. 사전심사(제7조②, 시행령 제13조). **경쟁입찰 성립 = 2인 이상 유효입찰**(시행령 제11조).

| 방법 | 핵심 |
|------|------|
| **일반경쟁** | 자격 충족자 전원 참가 — **기본** |
| **제한경쟁** | 고시금액 미만 지역제한·중기간경쟁·특수설비·실적 등 |
| **지명경쟁** | 잠재 10명 이하 → **5명 이상 지명**, **2명 이상** 승낙 |
| **수의계약** | 천재지변·비밀·1인 생산·**1억 이하**(청년창업 등 **5천만 이하**) / **2천만~5천만** 전자 2인 견적 |

**총액 vs 단가:** 총액계약(일회·수요 적음) / 단가계약(반복·정형) → **제3자 단가·다수공급자**(`조달사업법` 제12·13조).

**이행기간:** 당해연도 / **장기계속**(총사업 확정·예산 미확정·연도별) / **계속비**(총예산·총낙찰·연부액).

**대가확정:** **확정계약**(일반) / **개산계약**(전체 미확정·사후정산) / **사후원가검토조건부**(일부 비목만 미확정).

**계약인원:** 단독(1명) / **공동계약**(2인 이상, `국가계약법` 제25조).

### 1.5 낙찰자 결정제도 (시행령 제42~46조)

| 제도 | 요지 |
|------|------|
| **일반(적격심사)** | 예정가격 이하 **최저가 순** + 이행능력·일자리창출 등 (제42조①⑤) |
| **2단계경쟁** | 1단계 규격 → 2단계 가격; **단계별 2인 이상** (제42조③) |
| **종합심사** | 가격+능력+사회적 책임; 시설 **100억+**, 기본·실시설계·건설사업관리 등 **고액** (제42조④) |
| **협상에 따른 계약** | 기술80%+가격20% → 우선협상 (제43조·43조의2) |
| **경쟁적 대화** | 요구사항 불명확·복잡 시 대화 후 제안 (제43조3) |
| **종합낙찰** | 가격+총에너지소모 / 성능·환경 (에너지 6종·친환경 9종) (제44조) |
| **희망수량경쟁** | 최저가 순 **수요물량까지** 복수 낙찰 (제46조) |

**낙찰 원칙** (`국가계약법` 제10조): ①이행능력+최저가 ②평가기준 최유리 ③시행령 특별기준. **세입** → **최고가**. 공사 — 재료·노무·경비+VAT **98% 미만** 입찰 **낙찰 불가**(제10조③).

### 1.6 계약 체결·이행

- **계약서 7항목** (제11조): 목적·금액·이행기간·계약보증금·위험부담·지체상금·기타.
- **작성 생략** (시행령 제49조): **3천만 이하**, 경매, 즉시 인수 매각, 기관 상호계약, 전기·가스·수도 등.
- **계약보증금:** **10% 이상**(경기위기 고시 시 **5%**); **5천만 이하** 등 면제 가능 (시행령 제50조).
- **검사 후 대금** (제15조): 청구일 **5일 이내**(재난 고시 **3일**).
- **담보책임** (제17조): 공사 — `민법` **671조 이내**(토지·건물 **5년**, 석·금속 등 **10년**).
- **하자보수보증금** (제18조): **2~10%**; 3천만 이하 등 면제 (시행규칙 제72조).
- **물가변동** (제19조, 시행령 제64조): 체결 후 **90일+** · 조정률 **±3%** · 품목조정률 우선.

### 1.7 이의신청·분쟁조정

| 단계 | 기한·기관 |
|------|-----------|
| **이의신청 대상** | 추정 4억+ 종합공사 / 1억+ 전문공사 / 8천만+ 기타공사 / 5천만+ 물품·용역 (시행령 제110조) |
| **신청** | 행위일 **20일** 또는 안 날 **15일** 이내 → 중앙관서 장 (제28조②) |
| **처분** | 접수 **15일** 이내 (제28조③) |
| **재심(조정)** | 통지 **20일** 이내 → **국가계약분쟁조정위원회** (제29조) |
| **조정 기한** | **50일** 이내 (제31조①) |
| **화해 효력** | 조정 완료 후 **15일** 이의 없으면 **재판상 화해** 동일 (제31조③) |

## 절별 핵심 (제2절 민법)

### 2.1 공공계약과 민법

- 공공계약 = 사법상 계약; **특별규정 없으면** 사적자치·계약자유·민법 원리 **그대로**.
- **국가계약법 = 민법 특별법** (제3조).

### 2.2 계약 유형 분류

| 분류 | 공공계약 해당 |
|------|---------------|
| **전형 vs 비전형** | 전형(15종); 공공계약 **도급** 대표 |
| **쌍무 vs 편무** | 대부분 쌍무; **낙찰 = 편무예약**(본계약 전) |
| **유상 vs 무상** | 유상(매매 준용); 무상양여 = 증여 |
| **낙성 vs 요물** | 공공계약 **낙성**; 요물 = 계약금·보증금 등 |

### 2.3 성립 — 입찰과 민법 대응

| 절차 | 민법 |
|------|------|
| 입찰공고 | **청약의 유인** |
| 입찰 | **청약** |
| 낙찰 | **계약의 예약** |
| 계약체결 | **승낙** → 본계약 |

- **제527조** 청약 구속력 → 낙찰 후 청약 철회 제한.
- **제534조** 변경승낙 = **거절+신청약** → 협상계약 근거.
- **제528·529조** 승낙기간 경과 → 효력 상실; `국가계약법` **제28조** 수의계약 예외.

### 2.4 효력 — 쌍무계약

- **제536조** 동시이행항변권 / **제537조** 채무자위험부담 / **제538조** 채권자귀책.
- **제539~542조** **제3자를 위한 계약** ↔ `조달사업법` **제12조** 제3자 단가계약.

### 2.5 해제 vs 해지

| | 해제 | 해지 |
|---|------|------|
| **효과** | **소급** 소멸 | **장래** 소멸 (제550조) |
| **근거** | 이행지체(544)·이행불능(546) 등 | 계약·법정 해지권 |
| **공공계약** | 보증금 귀속 → 해제·해지 (시행령 **제75조**); 지체상금=보증금 → 해제 또는 **추가보증+유지** |

**부수적 채무** 불이행만으로는 **해제 불가**(주된 채무 필요, 대법원 2005다53705).

### 2.6 도급 vs 위임

| | 도급 (664~673) | 위임 (680~692) |
|---|----------------|----------------|
| **목적** | **일의 완성** | **사무 처리** |
| **보수** | **완성 후** | 업무 수행·기간 |
| **해지** | 특별 사유 | **언제든** (689) |
| **담보** | 하자 1년 / 토지·건물 **5·10년**(671) | — |
| **공공계약** | 제조·공사·용역 **도급** 성격 | 형식상 도급·실질 위임 시 판례 구분 |

### 2.7 화해 (731~732)

- **창설적 효력**; 착오 취소 제한 → **국가계약분쟁조정** 조정 = **재판상 화해** (제31조③).

## 절별 핵심 (제3절 조달사업법)

### 3.1 정책 기반

- **조달정책심의위원회** (제5조): 기재부 장관, **20명 이내** — 중장기 정책·공공수요·성과관리·혁신제품센터.
- **사회적·환경적 가치** 반영 (제6조).
- **조달의 날:** 매년 **9월 30일** (제8조).
- **조달통계** (제9조): 입찰·계약·대금·하도급; **나라장터·dBrain·e호조** 연계 의무.
- **자료요청** (제10조): 수출입·전자세금계산서 등.

### 3.2 계약 체결 요청 (`시행령` 제11조)

| 의무 위탁 | 기준(요약) |
|-----------|------------|
| 물품 | 국가기관 **1억+**(외국산 **20만 USD**) |
| 단가·MAS·제3자 | 국가·지자체 해당 계약 |
| 공사 | 추정 **30억+** 종합 / **3억+** 전문·전기·소방 |
| 협의 불가 | 수요기관 **직접** 계약 가능 (제11조④) |

- **제3자 단가계약** (제12조): 조달청이 **단가만** 선체결 → 수요기관 납품요구.
- **다수공급자계약** (제13조): **2인 이상** · **시장거래가격** 이하 유지 · 위반 시 **차액 감액·청구**.
- **특례** (제14조): 카탈로그·비축·**추정 20억+** 중기 **공동수급** 표준제품.

### 3.3 대금·수수료

- **대지급** (제15조): 단가·제3자·MAS·교통불편 지역·재해복구 등.
- **조달수수료** (제16조): **조달특별회계** 세입; 감면 **20% 초과 불가**.

### 3.4 공정성

- **불공정조달행위 6종** (제21조): 허위서류·직접생산 위반·원산지 허위·규격 불일치·MAS 가격 위반·우수물품 부정 지정.
- **거래정지** (제22조): **2년 이내** (MAS 가격·원산지 허위·세금서류 부정 등).
- **포상금** (제23조): 뇌물·불공정 **최초** 신고.

### 3.5 지원

- 수요기관 **SW·공사** 업무 지원·대행 (제24조).
- **우수조달물품·혁신제품**·해외진출 (제26~28조).

## 절별 핵심 (제4절 전자조달법)

### 4.1 전자공고·계약

- 경쟁입찰 전자 처리 시 **나라장터 공고** (제6조); 방법·시기 = **공공계약법** + GPA **특례규정**.
- **게시 vs 붙임 불일치** → **입찰공고문 우선** / **공고일** → **시스템 게시일** 우선.
- **전자계약 성립** (제9조): ①송신 ②상대 **동의** ③발주 **확정 송신**.

### 4.2 전자입찰서 (시행령 제5조)

- **동일 PC 1건**만 제출.
- 제출 후 **교환·변경·취소 불가** (중요 오류 → **취소 신청** 가능, 해당 입찰 **재참가 불가**; 재공고·재입찰 시 예외).
- 부속서류 **암호·바이러스** → **미제출** 간주.

### 4.3 전자공개 수의계약 (시행령 제6조)

- **2인 이상 견적** 수의 → **안내공고** + **전자견적서** (입찰공고·전자입찰서 용어 **사용 안 함**).

### 4.4 하도급 전자처리 (제9조의2)

- 청구·승인·지급 **전자화**; **전용계좌** 등록.
- 예: **하도급지킴이**, 대금e바로, 체불e제로. `건설산업기본법` **34조** — 3천만 미만·30일 미만 소규모 외 **의무**.

### 4.5 이용 제한 (제15·시행령 제10조)

- 허위신청·입찰질서 훼손·**3년 내 처벌**·**체납**·**부정당업자** 등.

## 시험 출제 포인트

1. **법체계 4+1** — 공공계약법(국가+지방)·조달사업법·전자조달법·계약사무규칙; **공법계약 배제**.
2. **국가계약법 = 민법 특별법** · 특수조건 > 민법 (판례).
3. **계약 원칙 4+1** — 신의성실·호혜·부당특약 무효·청렴계약.
4. **추정 vs 예정 vs 추정금액** · 국제입찰 = **추정가격 ≥ 고시금액**.
5. **예정가격 결정 4단계** · 복수예비가격 **15→4→평균**.
6. **계약방법** — 일반 원칙 / 제한·지명 **10명·5명·2명** / 수의 **1억·5천만·2인 견적**.
7. **경쟁입찰 2인** · **확정·개산·사후원가** · **장기계속 vs 계속비**.
8. **낙찰 7제도** + 시행령 조문 · **공사 98%** · 세입 **최고가**.
9. **계약서 7항·3천만 생략** · **보증금 10%(5%)** · **대금 5일(3일)** · **물가 90일·±3%**.
10. **이의 15·20·50·15** · **국가계약분쟁조정위원회**.
11. **민법** — 낙찰=**편무예약** · 청약유인·527·534 · **해제≠해지** · **도급671 5·10년**.
