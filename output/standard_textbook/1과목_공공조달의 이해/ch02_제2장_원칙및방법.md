# 제2장 공공조달 원칙 및 방법 (표준교재 핵심)

> PDF p.67–126 · 박문각 Part 2

## 한눈에 보기

- **조달 전 주기** — 요구사항 식별부터 계약 이행·사후관리까지 명확한 업무수행 기준·실행원칙 준수가 핵심.
- **5대 기본원칙** — 투명성, 비용 대비 가치(VFM), 경쟁성, 차별 금지·동등한 대우, 책임성 (GPA·UN·세계은행·OECD·EU 공통).
- **OECD 12대 실행원칙** — 투명성·청렴성·접근성·참여·효율성·균형성·전자조달·역량개발·통합·평가·위험관리·책임성 (기본원칙과 실행원칙으로 구분 가능).
- **조달 방법 분류** — 경쟁 여부에 따라 **경쟁적** vs **비경쟁적**; UNCITRAL 모델법 **10유형** → 유사성 기준 **5유형** 재분류.
- **국내 대응** — 일반경쟁(공개)·지명경쟁(선별)·제한경쟁 / 비경쟁은 **수의계약**; RFP 계열은 적격심사·경쟁적 대화·협상에 따른 계약 등으로 이행.
- **시험 주의**: 박문각 **「3대 기본원칙」**(경쟁성·비차별·투명성) vs 표준교재 **「5대」** — ch01·문제집 교차 함정.

## 절별 핵심 (제1절 원칙, 제2절 경쟁적, 제3절 비경쟁적)

### 제1절 공공조달 원칙

#### 1.1 공공조달 기본 원칙 (5대)

| 원칙 | 핵심 |
|------|------|
| **투명성** | 정책·계획·입찰공고·계약정보 등 공개(법적요건·기밀 제외); 공정성의 선행조건; 공개경쟁입찰이 투명성·청렴성 확보에 가장 효과적 |
| **비용 대비 가치(VFM)** | 최저가가 아님 — TCO·LCC·공급업체 역량·정책목표 종합; 생애주기 관점 지속가능성·비용효율·품질·효과성 조합 |
| **경쟁** | VFM·재정 효율 확보의 기본원칙; 투명성 달성 수단이자 청렴성 입증에 중요 |
| **차별 금지·동등한 대우** | 입찰·평가·계약 전 과정 동등 참여·공정 평가; UN·OECD·EU·WTO(GPA) 핵심; 불필요·비합리적 참가자격 금지 |
| **책임성** | 의사결정·결과·성과에 대한 성실 이행; 이해관계자 정보접근·이의제기; 접근성·평가·역량개발·참여가 구현 수단 |

#### 1.2 공공조달 실행 원칙 (OECD 12대, 2015)

**구조:** 실행 기반(전자조달·통합·역량개발) → 핵심 실행(투명성·청렴성·접근성·참여) → 목표(효율성·정책지원 효과성의 **균형성**) → 관리(위험관리·평가·책임성)

| 실행원칙 | 요지 |
|----------|------|
| **투명성** | 전 과정 공개·접근; 부족 시 부실·사기·부패·왜곡경쟁 위험 — 나라장터·조달데이터허브(120종+), 사전규격공개, 사후브리핑 |
| **청렴성** | 윤리·정직·규정준수; 부패·사기·이해상충 방지 — 행동강령·교육·내외부통제(청렴계약서 등) |
| **접근성** | 모든 규모 잠재경쟁자의 조달기회·정보접근; 지정정보처리장치(나라장터·S2B·NeaT·온비드 등) 게시 의무 |
| **균형성** | 1차 재정운영 효율 vs 2차 정책효과성; 예정가격 합리성·낙찰 일반원칙(국가계약법 시행령 제42조) — **우선구매·중소 목표**도 정책 효과성 측, **VFM·공정경쟁과 균형** 전제 |
| **참여** | 조달기업·지역사회·이해관계자 포용; 중소기업 참여 확대 정책 |
| **효율성** | 5R(Right Quantity·Quality·Time·Price·Place); 중앙조달·FA·MAS·전자카탈로그·동적구매·전자경매 |
| **전자조달** | 투명성·책임성·효율성·공정성 강화; KONEPS·차세대 나라장터 |
| **역량개발** | 담당자·공급업체 능력; 공공조달역량개발원·**공공조달관리사**(2025.3 국가기술자격 확정) |
| **평가** | 공정·투명·VFM 중심; 성과지표·피드백 |
| **위험관리** | 식별→평가→해결→모니터링→개선 5단계; BRIAS·담합통계분석 |
| **책임성** | 책임사슬·제재·이의제기·내외부통제 |
| **통합** | 재정·예산·전자정부와 연계; dBrain+·e호조·나라장터 |

**국내 투명성 제도 사례**
- 사전규격공개: 물품·용역 **추정가격 5,000만 원 이상** 의무; **절차 순서** — 사전규격 **공개** → 이견 **2주 이내** 검토·회신 → **입찰공고 반영** (`국가계약법` 하위예규, `지방계약법` 제9조의2)
- 청렴: `국가계약법` **제5조의2** 청렴계약서 의무, **제5조의3** 위반 시 해제·해지
- **재공고 후 수의**: 유찰·재입찰·재공고 후에도 무응찰 등 → **수의계약** 예외 가능(조건 임의 변경 불가 — ch06)

### 제2절 경쟁적 공공조달 방법

#### 2.1 공공조달 방법 개요

**UNCITRAL 모델법 10유형**
1. 공개입찰 2. 제한입찰 3. 견적요청 4. 협상 없는 제안요청 5. 2단계경쟁 6. 대화 기반 제안요청 7. 연속협상 제안요청 8. 경쟁적 협상 9. 전자적 역경매 10. 단독공급원조달

**5유형 재분류 (경쟁 유무·절차 속성)**

| 구분 | 경쟁적 | 비경쟁적 |
|------|--------|----------|
| 공개경쟁 | 공개입찰·제한입찰 | — |
| 제안요청서 | 협상無·협상기반·연속협상·경쟁적협상 | — |
| 2단계입찰 | 2단계경쟁 | — |
| 견적요청 | (복수 견적 가격경쟁 시) | 소액 직접구매 |
| 단독공급원 | — | 단독공급원 |

> **5유형 각주**: **선별(지명)입찰** = 교재 표30 「선별·제한입찰」 칸의 **경쟁적** 방법 / **제한입찰**(공개·선별 실패·자격 미달 등) = 「**제한경쟁**」 — 선별과 제한경쟁 혼동 주의.

- **견적요청(RFQ)** — 운영 방식에 따라 경쟅·비경쟁 **양쪽** 해당 가능.
- **제한입찰** — 합리적 제한요건이면 자격 업체 간 실질 경쟁 → **경쟁적** 방법.

#### 2.2 경쟁적 조달 방법

- **원칙:** 경쟁적 방법 우선; 비경쟁은 원칙 위배·저하 없는 범위에서 신속 조달 시 예외.
- **국내 3분류:** 일반경쟁(공개) · 지명경쟁(선별) · 제한경쟁.

**가. 공개입찰**
- 가장 대표적·우선 고려; 공개경쟁입찰 = 일반경쟁.
- 필수: 중립·명확 기술사양, 전 잠재입찰자 개방, 객관적 자격·평가기준, GPA 등 국제공고.
- 단점: 기간 장기, 형식주의, 기술사양 사전 확정 전제, 최저가 편향·혁신 제약 등.
- **2단계 공개입찰:** 1단계 기술제안 평가 → 2단계 가격제안.

**나. 선별적(지명) 입찰**
- 공개입찰과 유사하나 **자격 충족 업체만** 초대; 최소 **3개 이상** 업체 초대 일반적.
- 오남용 시 경쟁성·접근성·차별금지·동등대우 위배 위험.

**다. 제한적 입찰**
- 공개·선별로 적격자 없거나 자격 미달 시; 비상·보안·긴급·연속성·특수요건 등 예외.
- **Single Contracting** — 복수 후보 중 1사 선정 / **Sole Source** — 독점 공급원 1사.
- 국내: 제한경쟁입찰; 최고 수준 제한 = **수의계약**.

#### 2.3 경쟁적 입찰 절차 실행

**입찰 방법 선택 요소 (7가지):** 조달기관 임무·사업요구, 품목특성, 시장, 시급성, 기술수준, 최적 입찰기법, 담당자 역량.

**가. 제안요청(RFP) — 2봉투(Two Envelop)**
- 기술제안서 먼저 평가 → 최소합격자만 가격제안서 개봉 → 합산 최고점·우선협상.
- **선정 4유형:** QCBS(품질·비용) · QBS(품질) · FBS(고정예산) · LCS(최소비용)
- **UNCITRAL 3유형 ↔ 국내**

| 유형 | 내용 | 국내 대응 |
|------|------|-----------|
| 협상 없는 RFP | 협상 없이 평가·낙찰 | **적격심사** (덤핑 방지·이행능력) |
| 대화 기반 RFP | 입찰 전 대화로 요건 구체화 | **경쟁적 대화에 의한 계약** (전 측면 심도 논의) |
| 연속협상 RFP | 기술순위별 순차 가격협상 | **협상에 따른 계약** (지식기반사업 등) |
| 경쟁적 협상 | 다수와 협상 후 최종제안 평가 | — |

**나. 2단계입찰**
- 기술·가격 **분리 제출**; 입찰자가 기술사양·과업범위 정의에 참여.
- **유형 1:** 1단계 최고 기술안 동의 → 2단계 가격만 / **유형 2:** 1단계 토론으로 RFP 완성 → 2단계 기술+가격.
- 국내 **2단계경쟁입찰:** 유형 2만 적용; 단순 2단계(One Envelop) vs 규격·가격 **동시입찰**(Two Envelop); 각 단계 **2인 이상** 유효입찰 필요.

> **동시입찰 vs 2단계경쟁**: **동시입찰** = 규격·가격 **한 번에** 제출·단계별 2인 이상 / **2단계경쟁** = UNCITRAL **유형 2**·1단계로 RFP 완성 후 2단계 기술+가격 — 국내는 2단계경쟁만 유형 2 적용.

**다. 전자적 역경매(ERA)**
- 온라인 실시간 최저가 경쟁; 비용절감 기대 vs 품질·관계 저하·소규모 혁신기업 배제 우려.

### 제3절 비경쟁적 공공조달 방법

#### 3.1 비경쟁적 조달 개요

- 특정 공급업체 **직접 선택**; 신속·특화접근·비밀·검증된 관계 vs 과다비용·투명성·공정성 저하·공급의존.
- 활용: 시간 부족, 공익 긴급, 기술전문·맞춤용역, 특허·호환·지적재산, 기존 연속성.
- 국내 명칭: **수의계약** (견적요청서로 예산 범위 확인 등 형식적 유사).

#### 3.2 견적요청(RFQ)

- 상용품·표준규격·가격 중심; 입찰문서 없이 신속.
- **최소 3명 이상** 견적 요청(2명 이하 시 예외기록); 준비기간 **약 5일**; 제출 후 변경·협상 불가.
- 수요기관이 대상 업체 지정 → 기본 **비경쟁**; 3사 이상 가격비교 시 **제한경쟁 유사**.
- 계약 분할·임곗값 조작 방지 위해 금액·빈도 제한.

#### 3.3 단독공급원(수의계약) 및 소액구매

**단독 vs 단일**
- **단독(Sole Source)** — 공급 가능 업체가 **유일**
- **단일(Single Source)** — 다수 존재하나 **전략적으로 1사** 선택

**직접조달 적합 사례:** 천재지변·비상, 객관적 유일공급, 특허·호환·지재권, 연속성, 경쟁보다 직접이 객관적 유리, 법령 요건 충족 — **내용·형식·법적 정당성**·상위 승인·문서화 필수.

**소액구매(MPT)**
- 간소화 조달; 민간 단순구매 유사.
- UN: **5,000달러**(승인 시 **1만 달러**); 미국 FAR: 국내 **1만 달러**·국외 **3만5,000달러**.
- 국내: 정부구매카드·유류카드, **500만 원 내외**(기관별) 소모품 등.

## 시험 출제 포인트

1. **5대 기본원칙** 명칭·영문·정의 (특히 VFM ≠ 최저가, 경쟁↔투명성↔청렴성 연계).
2. **OECD 12대 실행원칙** — 기본(투명·청렴·효율·접근·책임) vs 실행(균형·참여·전자조달·역량·평가·위험·통합) 구분.
3. **UNCITRAL 10유형** 목록 및 **5유형 재분류** 표.
4. **국내 입찰 방법 대응** — 일반경쟁·지명경쟁·제한경쟁·수의계약; RFP 3유형 ↔ 적격심사·경쟁적 대화·협상계약.
5. **공개·선별·제한 입찰** 차이, 선별 시 **3사 이상** 초대, 제한 시 Single vs Sole.
6. **RFP 2봉투** 절차, QCBS·QBS·FBS·LCS 4선정방식.
7. **2단계입찰** 유형 1·2 차이, 국내는 유형 2만·단계별 2인 이상 유효입찰.
8. **견적요청** — 비경쟁 기본·3사 견적·5일·가격비교 시 경쟁적 성격; **RFQ 이중성**.
9. **비경쟁 장단점**, 수의계약 정당성 요건·문서화.
10. **투명성 강화** — 사전규격 5천만 원, 2주 회신, 조달데이터허브, 청렴계약서.
11. **균형성** — 1차 재정효율 vs 2차 정책목표; 예정가격·낙찰 일반원칙.
12. **소액구매** UN·미국·국내 임곗값; 구매카드.

## 함정·비교형

| 비교 | 구분 포인트 |
|------|-------------|
| **기본원칙 vs 실행원칙** | 5대(국제 공통) vs OECD 12대(실행·관리); 투명성·책임성은 양쪽 모두 |
| **VFM vs 최저가** | VFM = TCO·LCC·정책목표 종합 / 공개입찰은 최저가 편향 비판 |
| **공개 vs 선별 vs 제한** | 참가범위(전체 vs 자격충족 초대 vs 예외·재입찰); 선별 **최소 3사** |
| **Single vs Sole** | 복수 중 1사 선택 vs 공급원 **유일** — 제한입찰 내 직접계약 |
| **경쟁적 대화 vs 대화기반 RFP** | **전 측면** 심도 논의 vs **특정 기술·설계** 중심 대화 |
| **협상無 RFP vs 적격심사** | 기술제안서 없이 서류=정량 기술자료; 협상 없음 공통 |
| **연속협상 RFP vs 협상계약** | 기술순위→순차 가격협상; 1순위 불성립 시 차순위 |
| **2단계입찰 유형1 vs 2** | 최고 기술안 **동의 후** 가격만 vs 1단계 **RFP 완성** 후 2단계 |
| **견적요청 경쟁 vs 비경쟁** | 3사 이상 가격비교 vs 1~2사 지정·단일 견적 |
| **수의계약 vs 견적요청** | 형식상 견적 확인 유사하나 수의는 **경쟁 배제** 직접계약 |
| **단독공급원 vs 단일공급** | 유일 공급 vs 다수 중 1사 **전략 선택** |
| **ERA vs 일반경매** | **최저가** 제시(역경매) vs 최고가 제시 |
| **소액구매 vs 견적요청** | MPT 이하 간소구매·카드 vs RFQ 절차(3사·5일 등) |
| **박문각 3대 원칙** | **표준교재 5대** | 문제집=경쟁성·비차별·투명성 / 교재=VFM·투명·경쟁·책임 등(ch01 교차) |
| **ITB vs RFQ** | ITB=**Invitation to Bid**·국제 **공개입찰** 통칭 / RFQ=**견적요청**·가격 중심 |
| **동시입찰 vs 2단계경쟁** | 동시=규격·가격 **한 번에**·단계별 2인 / 2단계=유형2·1단계 RFP 완성 후 2단계 |
| **국제 5대 기본원칙** | **OECD 12 중 기본 5** | 전자=VFM·경쟁·차별금지 등 / OECD 12 기본=투명·청렴·효율·접근·책임 |
| **우선구매·중소 목표** | **균형성 원칙** | 정책 효과(2차) vs 재정 효율(1차) — 한쪽만 강조 오답 |
| **사전규격 절차** | 입찰공고 선행 착각 | **공개 → 2주 회신 → 입찰공고 반영** 순서 |
