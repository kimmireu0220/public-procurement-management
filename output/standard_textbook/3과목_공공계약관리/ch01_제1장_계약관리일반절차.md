# 제1장 계약관리 일반 절차 (표준교재 핵심)

> PDF p.5–98 · 박문각 Part 1

## 한눈에 보기

- **계약관리** = 낙찰·협상 이후 **법적·관리적** 이행 통제; 조달 전 과정(계획~피드백) 중 **계약 이후** 구간.
- **공공계약** = 대법원 기준 **사법(私法) 계약** + 국가·지방계약법 **특별규율**; 신의성실·부당특약 금지(국가계약법 제5조).
- **4절 구조**: ①계약관리 계획(원칙·기능·성립) → ②**변경**(물가·설계·기타) → ③**이행**(납품·보증금·지체·성과) → ④**종결**(검사·대금·지체상금·하자).
- **시험 핵심 클러스터**: 보증금 3종(입찰/계약/이행·하자), **물가변동 90일+3%**, **지체상금**, **해제≠해지**, 국고귀속 **선행**.

## 절별 핵심

### 제1절 효과적 계약관리 계획

**계약의 자유 (4요소)**  
체결·상대방 선택·내용 결정·방식의 자유 — 공익·공정성에 의한 **외적 제한**(국가계약법 제5조: 신의성실, GPA 호혜, **부당특약 무효**).

**계약관리 vs 계약행정**

| 구분 | 계약행정 | 계약관리 |
|------|----------|----------|
| 초점 | 구매 준비·완료 **행동** | 목표 달성 **통제·방법** |
| 성격 | 행정적 | 관리적(비용·일정 일치) |

**계약 5대 기능**: 증거 · 관리 · 위험전가 · 지불 · 동기부여(인센티브계약 IC).

**성립요건 (4)**: 의사표시 · 이행가능성 · 법적·사회적 타당성 · 확정성 → **민법** 적용.

**계약관리 6단계(표준교재)**: 계획 → 체결 → 이행 → 변경 → 종결 → 사후관리.

**분쟁해결 수단**: ADR(조정·중재) · 행정심판·소송 · **집행정지** · 헌법재판(위헌법률심판 등). 실무: 부정당·거래정지 등 **집행정지** 빈출.

### 제2절 계약변경 관리

**변경 3대 사유** (국가계약법 제19조): ①**물가변동** ②**설계변경** ③**기타 계약내용 변경**(천재지변·전쟁 등).

**변경계약 절차 (6단계)**  
(1) 가능 여부 판단 → (2) **합의각서** → (3) 변경 의뢰(방침서·산출내역 등) → (4) 나라장터 초안 송신 → (5) 보증·인지세 → (6) **변경계약 체결**(dBrain 등록).

**물가변동 조정 (시행령 제64조)**

| 요건 | 내용 |
|------|------|
| **기간** | 계약 체결일(장기=1차) 또는 **직전 조정기준일**로부터 **90일 이상** |
| **등락** | 입찰일(수의=체결일) 기준 **품목조정률 또는 지수조정률 ±3% 이상** |
| **동시 충족** | 두 요건 **동시** 충족일 = **조정기준일** |
| **재조정** | 조정기준일 후 **90일 이내 재조정 불가** |
| **특례** | 천재지변·원자재 급등 등 → **90일 미만**에도 조정 가능(제5항) |
| **단품슬라이딩** | 공사 특정 자재(공사비 5‰ 초과) **±15%** (제6항) |
| **방법** | 품목 vs 지수 **택1**, 계약서 명시; **이행 중 변경 불가** |

**조정신청**

- **증액**: 계약상대자 **신청 필수**(자동 조정 아님 — 대법원 2004다28825).
- **감액**: 발주기관이 조정내역서 **작성·통보**.
- **조정신청일**: 요건 증명 **관계서류 첨부 접수일**; 형식 미비 시 보완일(유권해석).
- **증액 시한**: 대가 수령 **전**; 청구 후 **30일 이내** 조정(규칙 제74조).
- **산식**: 증감액 = **물가변동적용대가** × (품목·지수 조정률); 선금 공제.

**설계변경·기타**: 공사에서 분쟁 비중 큼; 사정변경 원용. **덤핑수주·조건 미숙**으로 시공 거부 **불가**.

### 제3절 안정적 계약 이행 관리

**공급 7단계**: 수요 제기 → 구매계획 → 수요 상세화 → 공급자 발굴 → 선정 → 체결 → **계약관리**.

**납품 이행 점검**: 계약담당·품질보증공무원 — 자료 제출·현장 **수시·불시** 점검(MAS 급식 특수조건 사례).

**미이행 대응**: **해제·해지** · MAS **거래정지** · **계약보증금** · **지체상금**.

**계약보증금 (3.6절)**

| 항목 | 내용 |
|------|------|
| 성격 | 예정 **손해배상액** + 위약금적 기능 |
| 금액 | 계약금액 **10% 이상**(단가·장기=총액 기준) |
| 면제 | 공공기관·출자50%+법인·협조합 등, **5,000만 원 이하**, 관습상 부적절 등 |
| 면제 시 | **지급각서** 제출 |
| 국고귀속 | 의무 불이행·장기 2차 미체결 등(제12조) |
| 부분 이행 | 기성·기납 **인수분** 보증금 **제외** 귀속 가능 |
| vs 지체상금 | **불이행→귀속**, **지체→지체상금**; 귀속 시 지체상금 **병과 안 함**(유권해석) |
| vs 해제 | **국고귀속 선행** → 해제·해지(시행령 제75조; 보증 부종성) |

**지체상금**: 예정 **손해배상액**, 귀책 입증 **불요**(판례); 면제 사유 = 불가항력·발주 책임 등.

**성과관리**: 계약이행실적평가(MAS 별표4) · **SLA/SLM** · **PBSC**(성과기반 용역, 미국).

**MAS 이행실적평가**: 납기·품질·만족도·서비스·성실도 → **최우수~미흡** 4등급; 2단계경쟁·재계약 활용.

### 제4절 계약 종결 관리

**검사 vs 검수 vs 시험**

| 구분 | 주체·내용 |
|------|-----------|
| **검사** | 검사공무원 — 규격·법령 **적합** |
| **검수** | 출납담당 — 수량·손상 **확인** |
| **시험** | 시험기관 — 기능·성능 **측정** |

**검사 기한**: 이행완료 통지 후 **14일**(경기위기 고시 시 **7일**); 100억+ 공사 등 **연장** 가능.

**검사 면제**: KS·우수기업·품질관리능력 적합 등 — **생명·안전** 필요 시 예외.

**대금 지급**: 검사·검수 후 청구일 **5일 이내**(휴일 제외); 하도급대금 **15일 이내** 현금.

**납품대가 공제**: 기납·선금·**지체상금**·유보·**하자보수보증금**.

**지체상금 (4.3절)**

- 징수액 = **(계약금액 − 기납금액) × (지체일수 − 면제일수) × 지체상금률**
- 검사기간: **기한 내** 납품 시 검사기간 **제외**; 기한 후 제출 시 **다음날~검사통과**
- 성질별 **지체상금률** — 시행규칙 제75조(물품·용역·공사 구분)

**하자담보**: 민법 **6개월** vs 조달 **계약서 특기 하자기간**; 조달은 **과실 입증**으로 면책 가능.

**보증금 3종 (4.5절)**

| 종류 | 취지·시점 |
|------|-----------|
| **입찰보증금** | 입찰참가 **갈음**(지급각서); 낙찰 후 미체결 → **국고귀속** |
| **계약보증금** | 체결 시 **10%+** |
| **하자보수보증금** | 준공 후 **하자** 담보(계약서 명시) |

## 시험 출제 포인트

**계약 원칙·성립**

- 사법계약 + 공법 특별법; 제5조 **부당특약 무효**
- 계약 5기능 · 성립 4요건 · 민법 적용

**물가변동 (최빈)**

- **90일 + 3% 동시** = 조정기준일; 직전 조정기준일 기준 **재조정**
- 설계변경 후 90일 = **당초 체결일·직전 조정기준일** 기준(설계변경일 아님)
- 증액 **신청必要** / 감액 **기관 작성**
- 품목·지수 **혼용 불가** · 계약서 **사후 변경 불가**
- **90일 특례**(천재지변·원자재 급등), **단품슬라이딩 15%**

**보증금**

- 입찰(참가갈음) vs 계약(10%) vs 하자 — **목적·시점** 구분
- 국고귀속 **→** 해제·해지 순서; 부분 이행 **기성분 제외**
- 귀속 vs 지체상금 **병과 X**

**지체상금**

- 예정손해·귀책 불요; 검사기간 산입 규칙
- 공사: 지체상금 = **계약보증금 상당** 시 **해제·해지**(일반조건 제44조)

**해제·해지**

- **해제** = 소급(민법) · **해지** = 장래(원칙) — 1과목·3과목 교차
- **낙성/요물**: 본 장 미기재 → **민법 제565·567조**(계약금=해약금 추정) · 1과목 ch06 교차

**종결**

- 검사 14일/7일 · 대금 5일 · 하도급 15일
- 시운전 조건부 **성능이행보증금 15%**
