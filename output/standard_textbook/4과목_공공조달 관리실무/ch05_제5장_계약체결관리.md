# 제5장 공공조달 계약 체결 관리 (표준교재 핵심)

> PDF p.175–198 · 박문각 Part 5 · 계약·착수·보증

## 한눈에 보기

- **전자계약 흐름**: 발주기관 **계약서 초안** → 조달업체 **계약응답서**(+지급각서·청렴서약) → 발주 **계약체결통보서** = **확정**.
- **변경계약**: 업체 **변경요청서** → 승인 → 변경초안 → **변경응답서** → **체결승인통보**; 반려 시 **보완·재요청**.
- **착공·착수**: 공사=**착공계**, 물품·용역=**착수계** — **이행** 메뉴, **전자서명** 필수.
- **보증**: 국가계약법 **§12** — 계약보증금·**보증서**; **전자보증요청** 선택.
- **네트워크론**: 중소기업, **계약서 담보** 대출·**자동 상환**; **채권양도통보**.

## 절별 핵심

### 제1절 계약체결

**나라장터 메뉴**: **계약** → 계약체결관리목록

**12단위업무(물품·일반용역 공통 화면)**

- 발주: 초안 송부 → 업체: **응답서**·선제조건 확인 → 통보서

**조달업체 계약응답**

- 초안 **이의** → 반려사유 첨부 가능
- 무이의 → **지급각서·청렴서약서** → **확정·송신**

**변경계약**

| 단계 | 주체 |
|------|------|
| 요청 | 조달업체 **변경요청서** (사유·금액·기간·**지체상금** 면제 여부) |
| 승인/반려 | 발주기관 |
| 체결 | 변경초안 → 변경응답 → **승인통보** |
| 물품 | **납품기한** 일괄·개별 변경, **물품변경납기통보서** |

**지체상금**: 당초기간 **초과** 변경 시 — **귀책 없음·정당 사유** → **면제** 선택 가능

---

### 제2절 착수보고와 착수(착공)계 제출

- **이행** → 착공(착수)계: **착공일**·첨부 → **전자서명** → 발주 **접수·승인**
- **선금 청구**: 착수(착공) 후 가능
- **대지급**: 조달청 중앙조달·**MAS 납품요구** — 조달청 **선지급**(회전자금)
- **공동수급**: **대표업체** 선금청구 시 **타 구성원 확인** 절차

---

### 제3절 계약이행 보증 관리

**보증 요청**

- **이행** → 보증요청 목록
- 구성: 요청정보·대상·계약·업체
- 입력: **납부유형**, **전자보증**, 일괄보증, **시작·만료일**(계약과 **일치**), **보증금률·액**
- **변경계약** 기간 연장 → 보증 **연장·추가**

**보증기관**: 시행령 **§37** (보험·공제 등)

**네트워크론**

- 협약 **금융기관**에 계약정보 전송 → **무담보** 대출
- **채권양도통보** → 조달청, 대금에서 **상환**
