# 제2장 공급계획 수립 (표준교재 핵심)

> PDF p.63–92 · 박문각 Part 2 · 조달데이터허브·발주계획

## 한눈에 보기

- **공급계획** = 조달데이터허브(data.g2b.go.kr)로 **실적·예측** 수집 → **수요기관·시기** 선정 → **발주계획·사전규격** 선제 대응.
- **데이터허브 5대**: 데이터제공·시각화·**분석**(키워드·수요예측·입찰혼잡도·빅데이터)·고객지원.
- **입찰공고만** 보면 준비 **5~10일** — **발주계획**으로 **수개월~1년** 전 정보 확보.
- **타임라인**: 발주계획(연초) → 사전규격(5~10일) → 의견 반영(최대 **2주**) → 입찰공고(규격공개 **7일** 후).

## 절별 핵심

### 제1절 공급정보 수집과 분석

**조달데이터허브 개요**

- **통합 출처**: 나라장터·종합쇼핑몰·혁신장터·하도급·벤처나라·이음장터 + 자체조달·디브레인·방사청·한전 등
- **분류**: 조달대상물(물품·용역·공사) × 조달단계(발주·입찰·계약·이행)
- **보고서**: 120건+(2025.10, 비로그인); **정형**(MSTR) vs **오픈 API**(앱·웹 개발)
- **로그인**: 조달청·수요기관·조달업체·일반 **4유형** 차별 보고서

**대표 보고서**

| 보고서 | 활용 |
|--------|------|
| 소관구분별 수요기관 등록 실적통계 | **10개** 수요기관 유형별 기관 수 |
| 조달업체 등록 내역 | 세부품명별 **경쟁업체** |
| 나라장터 발주계획 내역 | **연간** 입찰 예정 |
| 입찰공고 및 계약내역 | 전년도 **패턴**·업체 |
| 입찰공고 내역 | **지역제한** 등 필터 검색 |

**데이터 시각화**: 지역·세부품명 **그래프** — 보고서/API는 **문자 중심**

---

### 제2절 조달데이터 분석 및 예측

**데이터 분석 4기능**

| 기능 | 내용 |
|------|------|
| **쇼핑몰키워드 분석** | 세부품명 검색량·기관별 납품실적·**급상승 TOP10** |
| **공공조달 수요예측** | 중앙 vs **자체조달**, 물품·용역·공사, 지역·계약방법·물품분류 |
| **입찰혼잡도 예측** | 주간·일·시간대 **6단계** — 과부하 회피 입찰 시각 |
| **빅데이터 실험** | **이미지→세부품명 추천**; MAS 품명 확인 |

**활용 논리**: 키워드·실적 **추세** + 보고서 **교차검증** (예측은 **오차 가능**)

---

### 제3절 공급대상 수요기관 선정

**발주계획 (나라장터 → 발주 → 발주계획)**

- 전년 예산 편성 → **차년도** 사업·**발주계획** 공개(지자체 계약정보시스템 등)
- 상세: 발주기관·**발주시기**·조달방식·계약방법·**예산액**
- **검색ON**: 발주계획·입찰공고·계약요청 **통합검색**

**사전규격 → 입찰 (사례: 청소용역)**

| 단계 | 시점·기한 |
|------|-----------|
| 발주계획 공개 | 예: 1월 (10월 발주 예정) |
| 사전규격공개 | 예: 10/22, 의견 **5일** (~10/27) |
| 입찰공고 | 규격 마감 **7일 후** (예: 11/3) |
| 의견 보완 필요 시 | 답변 **최대 2주** → 공고 **지연**·준비기간 **추가** |

**입찰공고 후 준비**: **5~10일** — 사전규격 단계 **5~10일** vs 발주계획 **수개월~1년**

**지역제한 예**: 일반용역(청소) **추정가격 5억 미만** — 입찰공고 내역 보고서로 **강원** 등 필터

**오픈 API**: 발주계획현황·계약정보·**낙찰정보** — 모바일 실시간

**입찰 실무 연계(교재 사례)**

- 적격심사, **낙찰하한율 87.745%**, 중기 간 경쟁·**지역제한**(3.3억 이하)
- 기초금액 **±3%** 복수예비가격 15건 → 예정가격

## 시험 출제 포인트

- **조달데이터허브 URL·5대 서비스** · 정형 vs API vs 시각화 vs 분석
- **공급계획 수립 절차**: 실적 수집 → 예측 → 수요기관 선정 → **발주계획** 선조회
- **발주계획 vs 입찰공고** 준비기간 — **서술형** 빈출
- **사전규격** 5일·7일·2주 함정
- **입찰혼잡도 6단계**, **120건+** 보고서
- **쇼핑몰키워드** → MAS 납품 시기·상품설명 키워드

## 함정·비교형

| 비교 | A | B |
|------|---|---|
| 데이터제공 | 정형 보고서 | 오픈 API |
| 분석 | 과거 실적 수집 | 수요·혼잡 **예측** |
| 준비 | 입찰공고만 | 발주계획+사전규격 |
| 조달 주체 | 중앙조달 | 자체조달 (수요예측 구분) |
| 청소용역 5억 | 지역제한 가능 | 5억 이상 일반 |
