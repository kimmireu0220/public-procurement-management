## 절별 핵심

### 제1절 다수공급자계약(MAS) 절차

**법·제도**

- 상용·표준화 **높은** 물품; 조달청-업체 **단가계약** → 수요기관 **납품요구**
- 업무 **6부문**: 공고신청·**적격성평가**·**사전심사**·가격협상·계약·검수

**MAS 체결 흐름**

1. 종합쇼핑몰 **구매입찰공고** 검색
2. **적격성평가** 또는 **사전심사** 신청(협상품목 등록)
3. 승인 → **가격 기초·가격자료** 제출
4. 조달청 **협상기준가**·**가격협상**
5. 계약서 초안 → **계약응답서**(보증 병행) → **계약통보**
6. **종합쇼핑몰 상품정보 등록**

**입찰공고**

- **10년** 원칙, **6개월 전** 재공고
- 규격서·공고서 — **최소 품질요건** 사전 충족 확인

**적격성평가 vs 사전심사**

| 구분 | 적용 |
|------|------|
| **적격성평가** | 사전심사 **14품목 제외** **전 물품** |
| **사전심사** | **14개** 품목(탄성포장재·난간·분리대·충격완충 등) — 적격성평가 **갈음** |
| 담당 | 신청서 → **종합지원센터** → MAS 담당부서 |

**신청서 구성**

- 심사·신청자 정보, **자기심사표**, **협상품목**(물품식별번호 **8자리**)
- **창업 3년 이내**: 신용등급 평가 **면제** 가능
- **납품실적** 증명(공고 요구 시)

**가격협상**

- 가격자료·**협상기준가** → 협상 성립 → 계약 (ch4 **가격협상** 동일 논리)

- **우수조달물품(우수제품)** — 조달청 지정; **기술개발제품 13종** 중 하나 (ch8)
- **목록화** 활용용도: **우수제품·MAS** 선택 (ch1)

*(교재: 우수제품 신청 화면·서류 — 조달청 **우수조달물품** 업무처리규정 연계)*
