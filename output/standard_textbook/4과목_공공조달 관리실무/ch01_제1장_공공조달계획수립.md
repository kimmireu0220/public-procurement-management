# 제1장 공공조달 계획 수립 (표준교재 핵심)

> PDF p.5–62 · 박문각 Part 1 · **실기** 나라장터 등록·품명등록

## 한눈에 보기

- **입찰참가자격 등록** = 공공조달 실무 **첫 단계**; 나라장터 미이용 수의계약 등에서도 **등록·계약상대자 요건**으로 요구.
- **근거**: 「국가종합전자조달시스템 입찰참가자격등록규정」; 등록정보는 **정당한 참가자격 증명** → 변경 시 **즉시 변경등록**, 허위·미변경 시 **자격 상실·입찰 무효**.
- **등록 흐름(7단계)**: 개인회원·본인인증 → 이용약관 동의 → 등록신청 → 온라인 서류 제출 → 현황 조회 → 승인/반려 (**관할 지방조달청** 검토).
- **4단계 업무**: 등록신청 → 신청 확인·안내문 인쇄 → 온라인 서류 제출 → 검토·승인; **20일 이내** 서류 미제출 시 **반려**.
- **조달대상물**: 공급물품 vs 제조물품 / 용역·공사·기타업종(면허·자격); **목록화** = 세부품명·품목(16자리) 신설.

## 절별 핵심

### 제1절 나라장터 사용자 등록

**등록 개요**

| 항목 | 내용 |
|------|------|
| 등록분야 | 물품·공사·용역 공급 사업자 |
| 메뉴 | www.g2b.go.kr → **이용자 관리** → 조달업체 |
| 승인 | **사업장 소재지 관할 지방조달청** |
| 완료 증명 | **경쟁입찰참가자격등록증** 출력 |

**등록신청 절차(실무 순서)**

1. **업체** 선택 → 조달업체 신규 등록
2. **이용약관** 동의 (취소 후 재신청 시 **약관 동의 재진행**)
3. **기본 정보**: 사업자·법인번호 **중복 확인**, 상호·주소·개업일·종업원 수 등
4. **대표자 정보**: 성명·주민번호, **실명인증**; 2인 이상 시 **대표 대표자** 1명 지정
5. **등록기관**(지방조달청) 선택, 담당자 정보
6. **사전동의서·청렴계약서** 동의 → **인증서 전자서명**
7. **안내문 인쇄** → 제출서류 **업로드** → **첨부 문서 제출** (20일 이내)
8. **승인요청** → 지방조달청 검토 → 승인/반려 (반려 시 **반려신청 건** 재신청)

**유의**

- 회사명·대표자명 **오류·미변경** → 참가자격 **무효**
- 정보 **사실과 상이** → 자격 상실·해지
- 변경사항 **지속 업데이트** 필수

### 제2절 공급대상물 유형별 품명 등록(물품목록화)

**입찰참가자격 — 조달대상물 등록**

| 유형 | 구분 | 실무 |
|------|------|------|
| **물품** | **공급물품** | 타 제조업체 물품 구매·공급 → **세부품명 찾기**(10자리) |
| | **제조물품** | 자체 생산·제조 → 세부품명 + **직접생산 확인** |
| **용역·공사** | 업종 등록 | **업종사항 찾기**, **면허등록번호** 입력 |
| **중복** | 물품+용역·공사 | 요건 충족 시 **복수 등록** 가능 |

**직접생산 확인 (제조물품)**

| 구분 | 등록 단계 | 비고 |
|------|-----------|------|
| 조달청 제조물품 | **2024.2~** 등록신청 시 **제조공정표 제출 생략** | 납품 실적 후 **실태조사**·공정표 제출 |
| **중소기업자 간 경쟁제품** | 등록신청 시 **직접생산 확인 증명서** **필수** | 중기부 지정 품목 |
| **융복합 제조물품** | 구성 세부품명 중 **1개 이상** 참가자격증에 **제조물품** 등록 | **OEM** → **공급물품** |

**목록화(목록정보시스템)**

- 세부품명 없을 때 → **품명·품목 신설**; **동일 품명·품목 중복 불가** → 사전 검색 필수
- **처리 기한(원칙)**: 품명 **24일**, 신산업 **17일**, 품목 **8일**, 품목 변경 **5일**
- **보완요청** 미응(15일) → **자동 반려**

**품목 등록(식별번호 8자리 → 물품목록번호 16자리)**

| 방법 | 조건 | 특징 |
|------|------|------|
| **개별요청** | 10개 미만(또는 일반) | 동의 → 요청자·활용용도 → 물품기본정보·이미지·속성 |
| **일괄요청** | **동일 세부품명 10개 이상** | 엑셀 양식, **최대 500건**; 양식 **임의 수정 금지** |
| **참조등록** | 기존·유사 품목 | 기존요청·검색화면·식별번호 참조 — **가장 수월** |

**개별등록 입력 핵심**

- **활용 용도**: MAS, 우수제품, 단가·총액계약, 벤처나라, 혁신시제품 등 (등록 후 **다른 용도 활용 가능**)
- **품목 구분 5종**: 일반용·계약용·시설자재용·입찰공고용·물품관리용 — **용도별 사용 범위 상이**
- **품목명 규칙**: 세부품명-제조업체명-모델명-대표속성값
- **이미지**: **실물 사진** 필수 (그래픽 X)
- **융합 vs 복합**: 융합=세부품명 결합 **단일 몸체**; 복합=독립 품목 **시스템** 구성

**물품분류 4단계**: 대(2)-중(4)-소(6)-세분(8) → **세부품명(10자리)**

## 시험 출제 포인트

**서술형·절차형 (고빈도)**

- **경쟁입찰참가자격 등록** 7단계·4단계 업무 흐름 **순서대로** 서술
- **공급물품 vs 제조물품** 정의·등록 차이·직접생산 서류 시점
- **중소기업자 간 경쟁제품** — **세부품명** 단위, 등록 단계 **증명서** 제출
- **목록화** 개별 vs 일괄(10개·500건), 처리일수, 15일 보완
- **품목 구분 5종** 용도 차이 (일반용 vs 계약용 vs 시설자재용 등)

**나라장터 실무**

- 메뉴: 이용자 관리 / 목록정보시스템
- **20일** 서류 제출, **승인=지방조달청**
- 변경등록 미이행·허위정보 → **무효·상실**

**박문각 Part1 교차** (`agent_extract/part1.md`)

- 발주기관 → **국가계약법 vs 지방계약법**
- LED 구매·설치 → **물품 vs 공사**·면허 검토
- **부정당업자** 사전 확인, **업종 vs 사업자등록 업종** 구분

## 함정·비교형

| 비교 | A | B | 구분 포인트 |
|------|---|---|-------------|
| 물품 | 공급물품 | 제조물품 | 구매·유통 vs 자가 제조 |
| 직접생산 | 조달청 일반 제조 | 중기 간 경쟁제품 | 등록 시 공정표 **면제** vs **증명서 필수** |
| 목록화 | 개별요청 | 일괄요청 | 10개 미만 vs **동일 세부품명 10↑** |
| 융복합 | 융합상품 | 복합상품 | **단일 몸체** vs **시스템** |
| OEM | 제조물품 등록 | 공급물품 등록 | **상표권자** → 공급 |
| 품목구분 | 일반용 | 계약용 | 일반관리+계약 vs **계약만** |
| 등록 | 사용자 등록 | 입찰참가 | 기능 등록 = **법적 참가자격** |
| 증명서 시점 | **등록 신청 시** 제출(중기 간 경쟁) | **입찰공고일** 기준 유효 | 등록 제출일≠공고일 — **공고일** 유효·세부품명 **일치** |
