# 학습 프롬프트

공공조달관리사 과목별 **설명 전용** 강의 프롬프트. 각 파일 내용을 **별도 채팅방**에 붙여 넣어 사용한다.

| 파일 | 과목 | Cursor 첨부 |
|------|------|-------------|
| `1과목_공공조달의_이해.md` | 필기 1과목 | `@교재/1과목_공공조달의 이해` `@output/ocr/1과목_공공조달의_이해` |
| `2과목_공공조달_계획분석.md` | 필기 2과목 | `@교재/2과목_공공조달 계획분석` `@output/ocr/2과목_공공조달_계획분석` |
| `3과목_공공계약관리.md` | 필기 3과목 | `@교재/3과목_공공계약관리` `@output/ocr/3과목_공공계약관리` |
| `4과목_공공조달_관리실무.md` | 실기 4과목 | `@교재/4과목_공공조달 관리실무` `@output/ocr/4과목_공공조달_관리실무` |

## 진행

- 1과목: Ch 0부터 (출제기준·목차 포함)
- 2~4과목: Ch 1부터
- `다음` / `Ch N` / `쉽게 다시` 로 진도 조절
- 문제·퀴즈는 `output/problem_book_final/` 문제집으로 별도 학습
