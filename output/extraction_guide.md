# 문제 추출 기준 (전 과목 공통)

> 1과목 상세 예시가 아래에 많지만, **규칙은 4과목 동일**하다. 재추출 playbook은 `문제집_제작_프롬프트.md` 참고.

## 시험·교재 매핑

| 필기/실기 | 과목 | 교재 slug |
|---|---|---|
| 필기 1 | 공공조달과 법제도 이해 | `1과목_공공조달의 이해` |
| 필기 2 | 공공조달계획 수립 및 분석 | `2과목_공공조달 계획분석` |
| 필기 3 | 공공계약관리 | `3과목_공공계약관리` |
| 실기 4 | 공공조달관리 실무 | `4과목_공공조달 관리실무` |

## 대상

- `교재/<slug>/Chapter N/page_*.jpg` 원본 이미지
- OCR 보조: 4과목 전체 (`output/ocr/<slug_with_underscore>/`, 예: `2과목_공공조달_계획분석`)
- 정확도 기준은 항상 원본 JPG. OCR은 보조·감사용
- 1과목 `Chapter 0` = 출제기준·목차 (문제 추출 제외)

## 포함할 문제 유형

- 이론 페이지 안의 `Check Q&A`, `Check O&A`, `V Check Q&A` 등 박스형 확인 문제
- `단원별 출제예상문제`
- `최종점검 OX 퀴즈`

## 제외할 내용

- `해설`, `정답`, `정답 및 해설`, 하단 정답표
- 손글씨 동그라미, 밑줄, 체크 표시 등 풀이 흔적
- 문제를 설명하는 이론 본문
- 표지, 목차, 학습목표, 키워드, 교재 활용 안내

## 정리 형식

- 객관식:
  `1. 문제문`
  `   ① 보기` … `④ 보기`
- OX: `1. 문제문 (O/X)`
- OCR이 애매하면 원본 이미지 기준 보정
- 각 문제 묶음 아래: `<!-- source: Chapter N/page_XXXX.jpg -->`

## 산출 경로

| 단계 | 경로 | 비고 |
|---|---|---|
| 추출본 | `output/agent_extract/<slug>/chapterN.md` | 문제+정답 |
| 최종 문제집 | `output/problem_book_final/<slug>/` | 정답 제거, MD+HTML |
| 검증 | `…/추출_검증.md`, `검토_요약.md`, `누락_후보_대조.md` | `tools/` 스크립트 생성 |
