# 공공조달의 이해 문제 추출 기준

## 대상
- `교재/공공조달의 이해/Chapter 1`부터 `Chapter 7`까지의 원본 이미지와 `output/ocr/공공조달의_이해` OCR 텍스트를 기준으로 추출한다.
- `Chapter 0`은 교재 안내/구성 설명이므로 실제 문제집 본문에서는 제외한다.

## 포함할 문제 유형
- 이론 페이지 안의 `Check Q&A`, `Check O&A`, `V Check Q&A` 등 박스형 확인 문제
- `단원별 출제예상문제`
- `최종점검 OX 퀴즈`

## 제외할 내용
- `해설`, `정답`, `정답 및 해설`, 하단 정답표
- 손글씨 동그라미, 밑줄, 체크 표시 등 풀이 흔적
- 문제를 설명하는 이론 본문
- 표지, 목차, 학습목표, 키워드, 교재 활용 안내

## 정리 형식
- 객관식:
  `1. 문제문`
  `   ① 보기`
  `   ② 보기`
  `   ③ 보기`
  `   ④ 보기`
- OX:
  `1. 문제문 (O/X)`
- OCR이 애매하면 원본 이미지를 기준으로 자연스럽게 보정한다.
- 원문 문항번호는 유지하되, 같은 섹션 안에서 OCR 누락/중복이 있으면 순차 번호로 바로잡는다.
- 각 섹션 시작에 원본 구분을 쓴다. 예: `### Chapter 01 단원별 출제예상문제`
- 각 문제 묶음 아래에는 검토용으로 HTML 주석 출처를 남긴다. 예: `<!-- source: Chapter 1/page_0013.jpg -->`
